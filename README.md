# Крутой бот - телеграм-бот для транслитерации ФИО 

Этот бот (@verycool_docker_bot) принимает сообщение, которое должно содержать ФИО, на кириллице и транслитерирует их на латиницу!

## Требования

Для работы с этим проектом вам необходимо иметь установленный Docker.

## Как запустить бот с использованием Docker

### 1 - Клонировать репозиторий:
Клонируйте данный репозиторий себе на ПК

### 2 - Создать Docker-образ:
Перейдите в директорию с проектом в своей IDE (например VS Code) и выполните команду для создания Docker-образа (image):
"docker build -t telegram-bot ." 
Команда вводится в терминале IDE в соответствующей папке проекта (вводится без кавычек) 

### 3 - Запустить контейнер:
Запустите контейнер в фоновом режиме:
"docker run -d --name telegram-bot-container telegram-bot"

### 4 - Проверка логов:
Выполните данную команду для отображения логов в консоли:
"docker logs telegram-bot-container"
Вы увидете информацию о запуске пулинга (работе бота), а также последующие логи при работе с ним

### 5 - Проверка работы бота:

Попробуйте отправить в бота ФИО на кириллице в формате "Фамилия Имя Отчество", и бот должен вернуть транслитерированное имя.
